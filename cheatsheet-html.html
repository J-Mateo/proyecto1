Guía rápida con etiquetas, estructura y ejemplos básicos de HTML.

Estructura básica

Todo documento HTML necesita una estructura mínima.


<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi página</title>
</head>
<body>
  <h1>Hola mundo</h1>
</body>
</html>

Metaetiquetas

Etiquetas importantes para SEO, responsive y redes sociales.

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Descripción clara">

<meta name="robots" content="index, follow">
<link rel="canonical" href="https://ejemplo.com/pagina">

<meta property="og:title" content="Título">
<meta property="og:description" content="Descripción">
<meta property="og:image" content="https://ejemplo.com/img.jpg">
Buenas prácticas
  • DOCTYPE siempre al principio.
  • meta charset lo más arriba posible.
  • Usa viewport para responsive.

Encabezados y texto

<h1>Título principal</h1>
<h2>Sección</h2>
<p>Párrafo normal.</p>

<strong>Importante</strong>
<em>Énfasis</em>
<mark>Highlight</mark>

<code>console.log("inline")</code>
<pre>Bloque preformateado</pre>
Tip
  • Usa un solo h1 por página.
  • Mantén una jerarquía lógica: h1 → h2 → h3.

Enlaces y navegación

<a href="https://ejemplo.com">Externo</a>
<a href="/ruta">Interno</a>
<a href="#seccion">Ancla</a>

<a href="https://ejemplo.com" target="_blank" rel="noopener noreferrer">
  Nueva pestaña segura
</a>

<nav aria-label="Principal">
  <ul>
    <li><a href="#inicio">Inicio</a></li>
  </ul>
</nav>

Imágenes y multimedia

<img src="img.jpg" alt="Descripción">

<picture>
  <source srcset="img.webp" type="image/webp">
  <img src="img.jpg" alt="Fallback" loading="lazy">
</picture>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

Formularios

<form action="/procesar" method="POST">
  <label for="email">Email</label>
  <input id="email" name="email" type="email" required>

  <label for="msg">Mensaje</label>
  <textarea id="msg" name="msg" rows="4"></textarea>

  <button type="submit">Enviar</button>
</form>
Recuerda
  • Conecta siempre label con for e id.
  • Usa required, minlength y pattern cuando sea necesario.

HTML semántico

<header>...</header>
<nav aria-label="Principal">...</nav>
<main>
  <article>
    <header>
      <h2>Título</h2>
      <time datetime="2024-03-15">15/03/2024</time>
    </header>
    <section>...</section>
  </article>
</main>
<footer>...</footer>

Accesibilidad (A11y)

<a href="#main">Saltar al contenido</a>

<img src="decoracion.jpg" alt="">

<button aria-expanded="false" aria-controls="menu">
  Menú
</button>

<div aria-live="polite">Mensajes de estado</div>
Importante
  • Prioriza HTML semántico antes que añadir ARIA.
  • Usa alt="" en imágenes decorativas.

Mejores prácticas

  • HTML válido e indentación consistente.
  • Imágenes con alt descriptivo.
  • Links externos con rel="noopener noreferrer".
  • Usa loading="lazy" en imágenes no críticas.
  • Comprueba tu web con Lighthouse, WAVE y Validator.
